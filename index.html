PokeDex

This React-based Pokedex project uses the PokeAPI and Axios to make HTTP requests for Pokemon data, and implements DOM manipulation to dynamically add, remove and modify the displayed information. The responsive and intuitive interface, built with React, allows for seamless navigation and a smooth user experience.

NutriTeam

This nutrition project harnesses the power of the Spoonacular API and JavaScript to provide users with detailed nutritional information for various food items. Users can input food items or recipes, and the API will provide data on calories, macronutrients, and micronutrients. The JavaScript frontend makes it easy for users to access and interpret this information, allowing them to make informed decisions about their diets and meal planning.
